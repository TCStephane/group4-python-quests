

# Python Quests – Group {Your Group Number}

This repository contains our solutions to the **Python Quests**, designed to build foundational programming skills in Python. Below, each quest includes a **context** and an explanation of the **logical approach** used.

---

## **Level 1: The Magic of Words and Numbers (Variables & Printing)**

### **Quest 1 – Your First Spell**

* **Context:** Introduce basic output using `print()`.
* **Logic:** Print a welcome message directly using `print()`. The key learning was the explicit command for the computer to display information.

### **Quest 2 – The Naming Ceremony**

* **Context:** Learn about variables.
* **Logic:** Stored a hero’s name and superpower in variables. Combined strings with variables using f-strings for dynamic output.

### **Quest 3 – The Treasure Chest**

* **Context:** Data types (string vs integer).
* **Logic:** Created variables for age (integer) and favorite food (string), then printed them together in a sentence. Ensured correct type usage.

### **Quest 4 – The Town Crier**

* **Context:** Combining variables and strings.
* **Logic:** Stored city name, year, and user name, then printed a dynamic message using f-strings to combine text with variables.

### **Quest 5 – The Echoing Cave**

* **Context:** Reassigning variables.
* **Logic:** Simulated a player’s health changing over time. Used arithmetic operations to update the variable and printed the final health.

---

## **Level 2: A Dialogue with the Machine (User Input & Basic Math)**

### **Quest 6 – The Fortune Teller**

* **Context:** Using `input()` to gather information from the user.
* **Logic:** Asked for user’s name and quest. Stored responses in variables and displayed a confirmation message.

### **Quest 7 – The Magic Number Converter**

* **Context:** Type conversion.
* **Logic:** Converted input strings to integers using `int()`. Calculated the user’s approximate age from their birth year.

### **Quest 8 – The Potion Brewer**

* **Context:** Basic arithmetic operators.
* **Logic:** Calculated the total cost of items by multiplying quantities by their unit costs and summing the results.

### **Quest 9 – The Greedy Goblin**

* **Context:** Modulo operator `%`.
* **Logic:** Determined how gold pieces are distributed among friends and calculated the remainder using `%`.

### **Quest 10 – The Architect’s Blueprint**

* **Context:** Floating-point numbers.
* **Logic:** Collected length and width as floats, multiplied them to calculate the area, and printed the result.

## **Level 3: The Crossroads of Logic (Conditional Statements)**

### **Quest 11 – The Guardian of the Bridge**

* **Context:** `if` statements.
* **Logic:** Checked if the user’s age was >= 18 and printed a message accordingly.

### **Quest 12 – The Two-Path Cave**

* **Context:** `if-else` statements.
* **Logic:** Implemented a simple password checker. Printed “Access Granted” if correct, otherwise “Access Denied.”

### **Quest 13 – The Maze of Many Choices**

* **Context:** `if-elif-else`.
* **Logic:** Converted a numerical score to a grade. Checked ranges in descending order to determine the correct output.

### **Quest 14 – The Logical Gatekeeper**

* **Context:** Logical operators (`and`, `or`).
* **Logic:** Verified multiple conditions at once. Only allowed entry if both conditions were satisfied.

### **Quest 15 – The Nested Riddle**

* **Context:** Nested `if` statements.
* **Logic:** Implemented multiple decision layers for a mini-adventure. The second decision depended on the first.

### **Quest 17 – The Endless Staircase**

## Notes

* **Quests attempted:** 1–24, 26–29
* **Key skills learned:** Variables, printing, input handling, arithmetic, loops, conditional logic, functions, and problem-solving.
* **Approach:** Each quest was approached logically:

  1. Understand the requirement.
  2. Identify the core concept (e.g., loop, function, conditional).
  3. Implement the concept in code.
  4. Test and iterate until correct output was achieved.
